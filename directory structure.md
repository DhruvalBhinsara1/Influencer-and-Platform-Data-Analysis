
# Project Directory Structure

This repository is organized for clarity and reproducibility. Below is the current structure:

```Practice Data Analysis/
│
├── data/
│   ├── messy_social_finance_dataset.csv    # Raw dataset
│   ├── cleaned_dataset.csv                 # Cleaned, analysis-ready data
│   └── category_platform_performance.csv   # Aggregated performance data
│
├── notebooks/
│   ├── 01_data_cleaning.ipynb              # Data cleaning & preprocessing
│   ├── 02_eda_visualization.ipynb          # Exploratory data analysis & visualization
│   ├── 03_trend_analysis.ipynb             # Trend analysis & advanced insights
│   └── results/                            # All generated plots and charts
│       ├── ad_spend_efficiency_monthly_trend.png
│       ├── anomalies_plot.jpg
│       ├── category_platform_performance.png
│       ├── correlation_heatmap.png
│       ├── distribution_plots.png
│       ├── engagement_by_campaign_duration.png
│       ├── engagement_vs_revenue_by_region_improved.png
│       ├── monthly_revenue_trends.png
│       ├── roi_by_platform.png
│       ├── top_10_brands_by_revenue.png
│       ├── top_10_categories_by_revenue.png
│       └── top_10_platforms_by_revenue.png
│
├── tasks_and_deliverables.txt              # Project stages and deliverables
├── directory structure.md                  # This structure overview
└── readme.md                              # Project documentation```

- All analysis and outputs are contained within the `data/` and `notebooks/` folders.
- Visualizations are saved in `notebooks/results/` for easy access and reporting.
- For any questions, please refer to the `readme.md` or contact the project maintainer.
